### Contributing to ORRO Protocol

Thank you for your interest in contributing to ORRO — the decentralized trust layer for creators.

ORRO is open-source from day one. We welcome code, documentation, bug reports, feature suggestions, and reviews.

### Code of Conduct

All participants are expected to follow the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
